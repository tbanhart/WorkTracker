Utilities

To support the core functions of the program, the WorkTracker application is packaged with a few additional utilites in order to make it easier to view and change application data.

Change User

As mentioned earlier, if a user enters the incorrect initials or wishes to change how their name is displayed on the QC sheets they generate, they can do so by selecting the Change User utility. The minimum and maximum size for entries are two and three, respectively. The initials a user designates are tied to their ID number provided by DST, which is derived from their Windows username. As such, a user changing to a different computer will still be known by the same initials if they change computers.

Clients

In order to designate a client to print QC sheets for, a user must first populate their client list through the use of the Clients utility. Clients are divided into prods, and a user may select which prod they are servicing to filter the client list into relevant clients. Beneath the boxes for all clients and user clients are buttons that allow a selected client from that list to be added or removed from the client list. When the user client list contains all clients the user is responsible for, clicking done will update the main window with all user clients.\nThe user client list is stored locally in the application data, and as such will be reset when the program updates, requiring the user to designate their clients again.
