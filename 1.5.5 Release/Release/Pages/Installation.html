Welcome to the WorkTracker Documentation

Installation

The WorkTracker files can all be setup from the Deferred Output folder of the I: drive. Since the setup is done for a shared folder, updates will be automatically detected when launching the WorkTracker application. When an update is detected by the launch sequence, a window will prompt the user to update the program. Always select yes, as a new update may patch in a fix for currently existing issues.

First Time Setup

The first time the application is used, a window will prompt the user to enter their initials. The initials entered will be shown on QC sheets that user generates when the submit the information to the printer via the main window. A minimum of two and a maximum of three characters can be entered for their initials, and if a mistake is made or the user wants to change them at a later date, they may do so by selecting the User utility in the toolbar of the main window. User initials are submitted to the DOP database and associated with the user's DT number from their Windows username. As such, when a user changes computers the program will automatically load their initials, provided they have used the program before.

Client Setup

Before printing the QC sheet for their first letter, a user will need to designate which clients they are responsible for. This can be done in the clients utility found in the toolbar and must be done before the user can begin printing QC sheets. For more information on selecting clients, see the Clients section of the Utilities chapter.
