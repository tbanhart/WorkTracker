Welcome to the WorkTracker Documentation

Core Features

The WorkTracker program was designed with the DOP in mind. The program is built to provide minor automation to help the process run more smoothly when generating QC sheets, recording work, and retrieving the status of a letter. The core automation processes for the program all revolve around this screen, the core WorkTracker UI. On this screen, a user is able to enter all information needed to fill out a QC sheet, and select whether they need a QC sheet printed, or are just needing to update a record in the database.

Correspondence Details

The Correspondence Details section of the window is the required information for the item to be submitted to the database. The client box is pulled from clients selected in the Clients utility. In the Letter Type box, the user may select which type should be shown on the QC sheet. If all information is not entered into these boxes and a user opts to print the QC sheet anyway, the program will refuse to continue until all information is complete.

Additional Information

All other information is handled in the Additional Information section. This includes the Name on a Death Certificate, the Tracking Number for overnight letters, and the corr type of the letter if other was selected for the type. Additionally, a user may add comments to a letter if they feel it needs information to be passed to the database. If the letter type is designated as other and the corresponding box is not complete, a window will prompt the user to enter the type in order to proceed.

Complete Letter

To send a letter to the database, a user will need to enter all requisite information in the form and select Print QC Sheet in the Complete Letter section. Selecting this button prints out a QC sheet automatically filled with all entered information. If a correspondence number already exists in the database upon submitting, the user will be prompted to update the current entry or start a new one; this feature allows a user to fix any mistakes they may have overlooked when first printing the QC sheet and improves accuracy.

Recent Letters

After a letter has been committed, it will automatically be added to the Recent Letters box. The box shows a list of the last three letters that have been added and allows the user to add all of the information from that letter back into the main window. After a letter has been selected, changing any of the information on the main area will apply the changes to the letter that was previously selected. Thus, if a mistake is made to the encorr number, type, or any other piece of information, it can be quickly remedied by selecting the letter and entering the correct information.

Toolbar

Features not essential to the core operations of the program can be accessed through the toolbar. This includes the options to change the user, select clients, and generate reports. Each of these features will later be discussed in greater detail. The toolbar also allows the user to exit the application.
